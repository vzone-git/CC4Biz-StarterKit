Claude Code for Business — Cheat Sheet

Click any prompt to copy it. Paste into Claude Code and replace the [bracketed] parts with your details.

🔍
Research
Deep Research
Research [topic]. Give me an executive summary, key findings with sources, implications for us, risks, and recommendations.
Competitive Analysis
Compare [Company A] vs [Company B] vs [Company C]. Pricing, features, target market, funding, traction, differentiators.
Market Sizing
Size the market for [product]. TAM/SAM/SOM with top-down and bottom-up approaches.
ROI Analysis
Build an ROI analysis for [investment]. Costs: [X]. Benefits: [Y]. Show conservative, base, and optimistic scenarios.
📊
Presentations
Pitch Deck
Create a pitch deck. We do [X], solving [problem]. Traction: [metrics]. Ask: [raise amount]. [paste notes]
Board Deck
Build a quarterly business review. Metrics: [paste]. Wins: [list]. Challenges: [list]. Next quarter plan: [list]. Decisions needed: [list].
Partnership Proposal
Create a presentation proposing a partnership with [partner]. Value for them: [X]. Value for us: [Y]. Key terms: [Z].

Investor Materials
Investor Memo
Write an investor memo for our [Series A]. Company: [name]. Traction: [metrics]. Market: [who]. Raise: [amount, use of funds].
Cold Outreach
Draft a cold email to [investor] at [fund]. Why they're relevant: [thesis/portfolio fit]. Our proof point: [best metric]. Ask: [meeting].
Financial Model
Build a financial model. Revenue: [X]. Growth: [assumptions]. Costs: [drivers]. Show 18-month monthly + 3-year annual, bear/base/bull.

Guardrails — Working Automatically in the Background
Final Folder Lock
Files in final/ can't be accidentally changed
Revision Guard
Prevents full rewrites when you asked for small edits
Main Branch
Blocks direct commits to main (if using git)
Confidentiality
Checks external docs for internal data leaks
Number Check
Catches conflicting metrics in documents
⚡
Quick Tips
Fill in CLAUDE.md Add your company, audience, and confidential terms. Less repeating yourself.
Paste raw material Notes, outlines, data, URLs. Claude works best with something to work with.
Revise, don't rewrite "Fix the intro" beats "redo the whole thing." Ask for surgical edits.
Use final/ folder Move approved work to final/. It's protected from accidental changes.
